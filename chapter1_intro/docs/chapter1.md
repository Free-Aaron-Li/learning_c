# 第 1 章 导言

## 1.1 入门
程序：函数+变量。

变量：存储计算过程中使用的值。

`main`，特殊函数名，程序的入口。

`<stdio.h>`，标准输入/输出库。

`"hello,world\n"`，字符串或字符串常量。

## 1.2 变量与算术表达式

程序中允许出现空格、制表符或换行符之处，都可以使用注释。

所有变量都必须先声明后使用。声明用于说明变量的属性，其由一个类型名和一个变量表组成。

`while`循环执行方式：

括号内条件为真 -> 循环体执行 -> ... -> 括号内条件为假 -> 循环结束。

> 建议
>
>每行只书写一条语句，并在运算符两边各加上一个空格字符。

`printf()`函数中`%d`用于指定一个整型参数，`%xd`中x用于表示打印宽度，例如：`%3d`。`%.xf`中x表示小数点后精度，例如：`%.2f`。此外，`%o`表示八进制数，`%x`表示十六进制数；`%c`表示一个字符；`%%`表示百分号本身。

> 💡
>
> 即使浮点常量取整型值，在书写时最好显式加上小数点，可以强调其浮点性质，便于阅读。

## 1.3 for语句

在允许使用某种类型变量值的任何场合，都可以使用该类型的更复杂的表达式。

`for`语句是一种循环语句，是对`while`语句的推广。

`for`语句执行顺序为：

初始化 -> 条件判断 -> 循环体 -> 条件判断 -> ... -> 条件判断 -> 循环结束。

## 1.4 符号常量

对于类似300、20这样的“幻数”，几乎无法向以后阅读该程序的人提供什么信息，并且对维护造成困扰。可以使用`符号名`（或称为`符号常量`）定义为一个特定的字符串。

符号常量名用大写字母拼写，很容易与用小写字母拼写的变量名相区别。

`#define`用于定义符号常量，且指令行末尾没有分号。

## 1.5 字符输入/输出

输入/输出按照字符流方式处理。

标准库提供`getchar()`和`putchar()`两个读/写一个字符的函数。

### 1.5.1 文件复制

EOF（end of file，文件结束）符号，定义在`<stdio.h>`中，是一个整型数。

### 1.5.2 字符计数

自增自减运算符（++、--），可以在变量名前或后，但++和--的优先级低于其他运算符。

### 1.5.3 行计数

单引号中的字符表示一个整型数，该值等于此字符在机器字符集中对应的数值，称之为*字符常量*。

### 1.5.4 单词计数

运算符`&&`比`||`高出一个优先级。

## 1.6 数组

`switch`语句提供编写多路分支程序的另一种方式，它特别适合于判定某个整型或字符表达式是否与一个常量集合中的某个元素相匹配的情况。
